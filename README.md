# blog

## Setup

### First Time

1. Install [mise](https://mise.jdx.dev/getting-started.html): `curl https://mise.run | sh`
1. Install tools: `mise install`

### Routinely

1. Install dependencies: `npm install`

## Development

Start the local development server:

```sh
npm run dev
```

## Test

Run the test suite:

```sh
npm run test
```

Build the site:

```sh
npm run build
```
